#### Пучок плоскостей.
---
Пучок плоскостей — это множество всех плоскостей, которые можно выразить как линейную комбинацию двух непараллельных плоскостей. Уравнение пучка плоскостей имеет вид:
$$\displaylines{
\lambda\pi_{ 1 } + \mu \pi _{ 2 } = 0
}$$
где $$\displaylines{
\pi_{ 1 }: A_{ 1 }x + B_{ 1 }y + C_{ 1 }z + D_{ 1 } = 0 \\
\pi_{ 2 }: A_{ 2}x + B_{ 2 }y + C_{ 2}z + D_{ 2 } = 0
}$$
заданные плоскости, причем их нормальные вектора неколлинеарные ${\displaystyle (\vec{n}_{ 1 } \cancel{||} \vec{n}_{ 2 })}$.

Доказательство:
Уравнение пучка в явном виде:
$$\displaylines{
 \lambda(A_{ 1 }x + B_{ 1 }y + C_{ 1 }z + D_{ 1 }) + \mu(A_{ 2}x + B_{ 2 }y + C_{ 2}z + D_{ 2 }) = 0 \\
 (\lambda A_{ 1 } + \mu A_{ 2 })x + (\lambda B_{ 1 } + \mu B_{ 2 })y + (\lambda C_{ 1 } + \mu C_{ 2 })z + (\lambda D_{ 1 } + \mu D_{ 2 }) = 0
}$$
Это линейное уравнение относительно ${\displaystyle x, \ y, \ z}$, а значит задает плоскость, если хотя бы один из коэффициентов при ${\displaystyle x, \ y, \ z}$ ненулевой.
Если ${\displaystyle \lambda \neq 0}$ и ${\displaystyle \mu \neq 0}$, то обе плоскости ${\displaystyle \pi_{ 1 }, \ \pi_{ 2 }}$ одновременно входят в пучок. В этом случае они пересекаются по прямой ${\displaystyle l}$, которая является общей осью пучка.
И уравнение прямой называется решение системы:
$$\displaylines{
l: \begin{cases}
 A_{ 1 }x + B_{ 1 }y + C_{ 1 }z + D_{ 1 } = 0 \\
 A_{ 2}x + B_{ 2 }y + C_{ 2}z + D_{ 2 } = 0
\end{cases}
}$$
У неё всегда есть решение, так как ${\displaystyle \vec{n}_{ 1 } \cancel{||} \vec{n}_{ 2 }}$.

Параметры ${\displaystyle \lambda, \ \mu}$ дают возможность изменять соотношение между плоскостями, что позволяет получить любую плоскость, проходящую через прямую ${\displaystyle l}$.
Если ${\displaystyle \lambda = 0}$, то пучок состоит из всех плоскостей, пропорциональных ${\displaystyle \pi_{ 2 }}$.
Если ${\displaystyle \mu = 0}$, то пучок состоит из всех плоскостей, пропорциональных ${\displaystyle \pi_{ 1 }}$.

Множество плоскостей, проходящих через точку ${\displaystyle M_{ 0 }(x_{ 0 }, \ y_{ 0 }, \ z_{ 0 })}$ называется связкой плоскостей с центов в точке ${\displaystyle M_{ 0 }}$.
Уравнение связки имеет вид:
$$\displaylines{
A(x-x_{ 0 }) + B(y-y_{0}) + C(z-z_{0}) = 0
}$$
где ${\displaystyle A, \ B, \ C \in \mathbb{R}, \ (A, \ B, \ C) \neq (0, \ 0, \ 0)}$, если ${\displaystyle \pi_{ 1 }, \ \pi_{ 2 }, \ \pi_{ 3 } \dots \in M_{ 0 }}$ и ${\displaystyle \pi_{ 1 } \cap \pi_{ 2 } \cap \pi_{ 3 } \dots = M_{ 0 }}$.
Уравнение связки — это уравнение пучка, где ось пучка (прямая пересечения) вырождается в точку.

Уравнение любой плоскости из связки:
$$\displaylines{
\sum_{i=1}^{3} \lambda_{ i }(A_{ i }x + B_{ i }y + C_{ i }z + D_{ i }) = 0
}$$
где ${\displaystyle \lambda_{ i } \in R}$, ${\displaystyle \pi_{ i }: A_{ i }x + B_{ i }y + C_{ i }z + D_{ i } = 0}$
